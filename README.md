# Frontend-Agenda-de-Contactos

Tema: Agenda de Contactos

Instruções de teste:

  1 - Comece por criar um contacto preenchendo os campos;
  
  2 - Após criar um contacto poderá premir no botão "Editar" para fazer as mudanças que desejar no contacto;
  
  3 - Abaixo também poderá adicionar notas a contactos existentes, podendo apagá-las posteriormente;
  
  4 - Caso um contacto deixe de ser de seu interesse poderá premir no botão "Apagar" para que ele seja removido inteiramente da lista de contactos.
